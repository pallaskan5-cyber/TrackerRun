# Mileon - Running Social Network

Mileon is a Strava-inspired running application that allows users to track their runs, connect with friends, compete in challenges, and analyze their performance.

## Features

- GPS run tracking with interactive maps
- Social feed to share activities with friends
- Advanced statistics and analytics
- Weekly challenges and leaderboards
- Dark mode support
- Fully responsive design

## Tech Stack

- **Frontend**: Next.js (Pages Router), Tailwind CSS
- **Backend**: Next.js API Routes
- **Database**: PostgreSQL with Prisma ORM
- **Authentication**: NextAuth.js with JWT
- **Maps**: Leaflet.js for interactive maps
- **Charts**: Recharts for data visualization
